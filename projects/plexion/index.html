PleXion Framework

PleXion is a powerful, modular Java framework built to rival Spring — but with less magic and more explicit architecture. It’s clean, extensible, and built for developers who want full control.

  • Modular core with plugin lifecycle hooks
  • Custom class loader with secure .jar scanning
  • Built-in REST support (Undertow → Netty)
  • Pluggable web features: JSON, DB (JPA/Hibernate), Docs, MVC, GraphQL
  • Native-image/GraalVM compatibility roadmap
